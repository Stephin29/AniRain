## 🌧️ What is AniRain?
AniRain is a desktop-first streaming application that aggregates content from across the web into one seamless, ad-free interface. No more paying for five different subscriptions.

- 🎌 **Anime-First:** Massive library with auto-tracking via AniList/MAL.
- 🐍 **Python Powered:** High-speed scraping and metadata processing.
- 💻 **Desktop Native:** Built with Electron for a premium Windows, macOS, and Linux experience.
- 🛡️ **Privacy Centric:** No telemetry, no ads, no tracking.

## 🛠️ Tech Stack
- **Frontend:** Electron / [React or Vue]
- **Backend/Logic:** Python (Scraping & Data Processing)
- **Communication:** [ZeroRPC / Socket.io / Flask-SocketIO] for Electron-Python bridge.
- **Styling:** Tailwind CSS
